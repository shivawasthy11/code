# Capinity Website

A responsive and modern company website developed to enhance user experience, improve accessibility, and provide a clean digital presence for Capinity. The project focuses on responsive web design, optimized layout structure, and improved visual presentation.

## 🚀 Features

- Responsive website design
- Modern and clean user interface
- Improved user experience and navigation
- Cross-device compatibility
- Optimized website layout and performance
- Structured and maintainable codebase

## 🛠️ Technologies Used

- CSS3
